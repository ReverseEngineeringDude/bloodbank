About This Project

The Online Blood Bank Management System is a college demo project designed to show how technology can facilitate blood donation and help save lives. This platform allows users to search for blood donors based on blood group and location.

  • Search donors by blood group and location
  • View detailed donor information
  • Mobile-friendly and responsive design
  • Quick contact options for urgent needs

Note: This is a demonstration project with sample JSON data for educational purposes.
